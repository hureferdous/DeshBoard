# DeshBoard
THE COMPONENTS OF OUR DASHBOARD PROJECT AND THEIR FUNCTION:  D3.js: A JavaScript based visualization engine, which will render interactive charts and graphs based on the data. Dc.js: A JavaScript based wrapper library for D3.js, which makes plotting the charts a lot easier. Crossfilter.js: A JavaScript based data manipulation library that enables two way data binding. Queue.js: An asynchronous helper library for JavaScript. Mongo DB: NoSQL Database used to convert and present our data in JSON format.
